# 🛕 Temple Locator

Temple Locator is a Python Full Stack Web Application that helps users discover temples across India.

## Features

- Search temples by city
- View temple details
- Google Maps integration
- Temple information display
- Responsive temple-themed UI

## Tech Stack

- Python
- Flask
- MySQL
- HTML
- CSS
- Git & GitHub

## Future Enhancements

- Temple images
- State-wise filtering
- Nearby temple recommendations
- User authentication
